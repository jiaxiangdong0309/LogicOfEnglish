# 快速启动指南 - 语音播放功能

## 🚀 三步开始使用

### 步骤 1：配置科大讯飞凭证

编辑项目根目录的 `.env` 文件：

```bash
# 打开文件
open .env
```

填入您的科大讯飞凭证（从 https://www.xfyun.cn/ 获取）：

```env
VITE_XFYUN_APPID=your_app_id_here
VITE_XFYUN_API_KEY=your_api_key_here
VITE_XFYUN_API_SECRET=your_api_secret_here
```

### 步骤 2：启动开发服务器

```bash
pnpm dev
```

### 步骤 3：测试功能

1. 打开浏览器访问 http://localhost:5173
2. 找到音标卡片
3. 点击展开任意卡片
4. 点击 🔊 图标播放音标或单词

## 🔧 故障排查

### 问题：点击没有反应

**解决方案**：
1. 打开浏览器控制台（F12）查看错误信息
2. 检查 `.env` 文件是否正确配置
3. 确认网络连接正常

### 问题：提示"播放失败，请检查网络连接或配置"

**解决方案**：
1. 验证 `.env` 中的三个变量都已填写且无多余空格
2. 确认科大讯飞凭证有效且服务已开通
3. 检查浏览器控制台的详细错误信息

### 问题：音频无法播放

**解决方案**：
1. 确认浏览器支持 Web Audio API（Chrome/Edge/Firefox/Safari 现代版本均支持）
2. 检查浏览器是否静音
3. 尝试手动与页面交互后再点击播放（某些浏览器要求用户交互后才能播放音频）
